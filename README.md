# ATOM-DNN Snippets

Snippets + AutoCompletions to add common DNN ( **DotNetNuke CMS** ) tags when building template skins in Atom Editor. Feel free to add your own.

**Lookups**
- NewsArticle Cheatsheet TAGS = **CTRL + 1**
